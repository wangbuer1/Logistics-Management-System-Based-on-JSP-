# 基于jsp的物流管理系统4



## 1、项目介绍

基于JSP的物流管理系统4有管理员和用户两种角色

管理员：会员管理、公告管理、车辆管理、货物管理、动态管理等

用户：登录、注册、找回密码、发布货物/车辆信息等


## 2、项目技术

后端框架：无

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.5-5.7、8.x版本都可） 
- maven项目：否
- 硬件环境：Windows
- **此项目基于JSP开发，JSP代码量95%**
